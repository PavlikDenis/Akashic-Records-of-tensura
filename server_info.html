Mod Information

Tensura Mod - The Core Experience

Our server runs on the powerful Tensura mod as its foundation, bringing the anime's world to life in Minecraft. This mod introduces:

  • Race System: Choose from Demon, Human, Spirit, or Divine races, each with unique abilities
  • Skill Trees: Extensive skill progression with branching paths
  • Transformation System: Unlock powerful forms as you grow stronger
  • Equipment System: Craft and upgrade legendary gear

Enhanced with Premium Addons

We've expanded the base Tensura mod with carefully selected addons to create the ultimate experience:

  • Mystecism - Advanced skills tree
  • Ani Tensura - Different abilities
  • Ash of Sin - Dark powers
  • BTR Ultima - Ultimate powers
  • No Game No Life - Anime powers
  • Tensura True Dragons - Dragon forms

    Launch Minecraft with the Akashick modpack
  3. 3
    Click "Multiplayer" then "Add Server"
  4. 4
    Paste the server address and click "Done"
  5. 5
    Select the server and click "Join Server"
